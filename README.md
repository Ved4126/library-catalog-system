# 📚 Library Catalog System

A Java-based console application that simulates a library management system with efficient search and book handling features.

---

## 🚀 Features

* Search books by:

  * ISBN
  * Title
  * Author
  * Category
* Display all books
* Display books by category
* Check out and return books
* Performance testing for search operations
* Sample data generation

---

## 🧰 Tech Stack

* **Language:** Java
* **Concepts:** Object-Oriented Programming (OOP)
* **Data Structures:** Arrays, Lists, Custom Objects
* **Tools:** VS Code, Git, GitHub
* **Runtime:** Java SE (JDK)

---

## ▶️ How to Run

### Option 1: Compile and Run Manually

```bash
javac *.java
java LibraryCatalogSystem
```

### Option 2: Use Script

```bash
bash compile_and_run.sh
```
